# 웹사이트의 목적
* 웹사이트를 보고 모임에 참석하고 싶게 한다. -> 다음 모임 공지 표시
* 우리가 그동안 만든거 자랑한다 -> 깃헙 저장소 + 설명
* 슬랙 들어오게 만든다 -> 슬랙 통계
* 모임에 안와본 사람이 보고 당황하지 않아야 함 -> 예쁜 디자인



# 개발
- yarn install
- yarn run gatsby develop
- 브라우져로 http://localhost:8000/
